# NAME

ogc-ls - List nodes in your inventory

# SYNOPSIS

**ogc ls** \[OPTIONS\]

# DESCRIPTION

List nodes in your inventory

# OPTIONS

**\--by-tag** TEXT

:   List nodes by tag

**\--by-name** TEXT

:   List nodes by name, this can be a substring match

**\--output-file** TEXT

:   Stores the table output to svg or html. Determined by the file
    extension.
